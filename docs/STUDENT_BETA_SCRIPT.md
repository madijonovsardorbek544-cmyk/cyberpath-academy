# Student Beta Script: 30-Minute Test

## Before you start

Use only fictional labs and demo data. Do not test real websites, accounts, networks, or classmates.

## Minute 0–5: Start

1. Open CyberPath Academy.
2. Create an account or use the provided demo login.
3. Read the beta banner and safety notice.
4. Say out loud what you think the product is for.

## Minute 5–10: Onboarding

1. Complete onboarding.
2. Choose your goal and level.
3. Answer diagnostic questions.
4. Submit onboarding feedback if anything is unclear.

## Minute 10–18: First learning loop

1. Open the recommended first lesson.
2. Read enough to answer what the lesson teaches.
3. Mark it complete.
4. Complete one practice question or review session.

## Minute 18–25: Lab and artifact

1. Open a safe beginner lab.
2. Submit answers using only the fictional evidence shown.
3. Create one portfolio artifact from the lab.

## Minute 25–30: Feedback

Answer:

- What confused you?
- What felt useful?
- What felt too easy or too hard?
- Would you recommend this to a classmate?
- Would you or your school pay for this if improved?
- What broke?
