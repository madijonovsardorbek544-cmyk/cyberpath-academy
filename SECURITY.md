# Security Policy

## Supported scope

CyberPath Academy is an educational platform focused on defensive cybersecurity learning in safe, fictional, or authorized contexts.

## Reporting a vulnerability

Please do not disclose vulnerabilities publicly before maintainers have a chance to investigate.

Report issues to: **security@cyberpath.local**

Include:

- affected route or page
- reproduction steps
- impact
- proof of concept if safe to share
- suggested remediation if known

## Out of scope

The following should never be added as product features:

- credential theft workflows
- malware deployment guidance
- persistence or evasion guidance
- live-target exploitation guidance
- unauthorized attack playbooks
